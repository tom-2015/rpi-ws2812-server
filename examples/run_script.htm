In this runs external script my_script.sh with the system command.

By default the system command is disabled, you must start the ws2812svr program with the -s parameter!
sudo ./ws2812svr -s
If you run it as a service, you must edit the /etc/ws2812svr.conf file and change: allow_system_cmd=1

setup 1,30,3 init fill 1,FF0000 render system ./my_script.sh fill 1,00FF00 render delay 1000 fill 1,0000FF render delay 1000
