You can render PNG,JPG and (animated) GIFs directly on your LED screen. Don't forget to compile with JPG support (default) on if you want to render JPG files. In this example an 8x8 display is used to first render a PNG and then render animated gif test.gif.

setup 1,64,3 init brightness 1, 32 config_2D 1,8,8,3,8,8 draw_image 1,2D/test.png render draw_image 1,2D/test.gif

NOTE: use double quotes (") if your file name contains spaces or a , chatacter.
draw_image 1,"path to file with spaces or ,"
